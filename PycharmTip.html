Pycharm 단축키 모음

1. Ctrl + Shift + Up : 코딩 단위를 위로 올리거나 내릴 수 있음

2. Ctrl + Shift + j : 여러줄을 한 줄로 만들거나 불필요한 부분 없애줌

3. Ctrl + p : 메서드에 필요한 매개변수 보여줌

4. Ctrl + E : 가장 최근에 작업한 파일 목록 보여줌

5. Ctrl + Space : 자동완성 추천기능

6. Alt + Up/Down : 한줄씩 위아래로 커서 움직일 수 있음

7. Ctrl + Q : Documentation 열어서 보여줌

8. Ctrl + B : 현재 프로젝트 내에서 해당 선언 어디어디 있는지 보여줌

9. Ctrl + Alt + Shift + n : 현재 프로젝트 내에서 쓰이는 네임스페이스 검색창

10. Ctrl + c : 자동으로 해당라인 전체 복사함

11. Alt + Enter : 오타 치면 밑줄 뜨는데 이 오타의 대안이 될 만한 단어 보여줌

12. Ctrl + Slash : 해당 형식에 맞는 한줄 주석 표시생성

13. Ctrl + Shift + Slash : 해당 형식에 맞는 여러줄 주석 표시생성

14. Alt + Home : 경로 표시 바로 이동함. 키보드로 경로 파일 선택가능

15. Ctrl + D : 해당 블록을 바로 복사

16. Alt + Enter : 빠른 수정 등 전구 뜰때 수정관련 아이템 띄워줌

17. Ctrl + Click : 원본 파일 열어줌 (객체나 함수 정의된 원본파일)

18. Ctrl + Shift + N : 긴 패스 파일 찾는 방법인데, 폴더명의 맨 앞글자/ 만 입력해도 폴더잡음

19. Shift (X2) : 최근 파일, 셋팅, 심볼 등 검색창

20. Ctrl + Alt + Shift + T : 리팩토링 메뉴 불러옴

  1. Move to Another Changelist : 로컬 체인지 창에서 특정파일 왼쪽클릭하면 메뉴 있음.
  2. Console 창에서도 자동완성이나 필요한 인수 정보 볼 수 있은 (Ctrl + space , Ctrl + Q)
  3. Todo list : 앞으로 해야할 일 관리 할 수 있도록 도와주는 기능임 - # todo 로 해서 정의하면 됨
